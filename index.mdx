Welcome to the meCash API! Our RESTful API provides the tools to build sophisticated financial applications, from processing instant global payouts to managing multi-currency digital wallets.

This page guides you through the essential first steps and core concepts for a successful integration.

## Quickstart Guide

Follow these four steps to get your account configured and ready to build.

<Columns cols={2}>
  <Card 
    title="1. Create Your Account" 
    icon="user-plus"
    href="https://docs.me-cash.com//account-creation"
  >
    Sign up for a meCash Business account to get access to your developer dashboard.
  </Card>
  <Card 
    title="2. Get Your API Keys" 
    icon="key"
    href="https://docs.me-cash.com//authentication"
  >
    Find your secret and publishable keys to authenticate API requests.
  </Card>
  <Card 
    title="3. Whitelist IP Addresses" 
    icon="shield-check"
    href="https://docs.me-cash.com//ip-whitelisting"
  >
    For security, all API calls must come from a whitelisted IP address.
  </Card>
  <Card 
    title="4. Set Up Webhooks" 
    icon="webhook"
    href="https://docs.me-cash.com//webhooks-event"
  >
    Configure an endpoint to receive real-time notifications for events.
  </Card>
</Columns>

Once your setup is complete, you're ready to start building.

➡️ **[Explore the API Reference](/api-reference)** to see all available endpoints.


## Before Your First Call
Keep these two critical concepts in mind.

<CardGroup cols={2}>
  <Card title="Authentication" icon="lock">
    All requests must include an `Authorization` header with your secret key.
    
    `api-key: YOUR_API_KEY`
  </Card>
  <Card title="Environments" icon="arrows-split">
    Use `sandbox.api.me-cash.com` for testing and `api.me-cash.com` for live data. API keys are separate for each.
  </Card>
</CardGroup>
