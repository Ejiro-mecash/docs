## Get started with meCash API in 4 steps

Follow this quickstart guide to integrate meCash API into your application.

### Step 1: Create Your Account

<AccordionGroup>
  <Accordion icon="user-plus" title="Sign up for meCash Business">
    <Frame>
      <img
        src="public/images/buisness-page.png"
        alt="meCash Business sign-up screen highlighting the create account button"
        style={{ borderRadius: '0.75rem' }}
      />
    </Frame>

    1. Visit [meCash Business](https://business.me-cash.com)
    2. Click **Sign Up** and complete the registration form
    3. Verify your email address
    4. Complete KYB (Know Your Business) verification
    
    <Tip>KYB verification is required to access API services.</Tip>
  </Accordion>
  <Accordion icon="shield-check" title="Complete KYB Verification">
    Submit the required documents:
    - Business registration certificate
    - Valid ID of business director
    - Proof of address
    
    <Warning>Your API access will be enabled after KYB approval.</Warning>
  </Accordion>
</AccordionGroup>

### Step 2: Get Your API Keys

<AccordionGroup>
  <Accordion icon="key" title="Generate API Key">
    <Frame>
      <img
        src="public/images/api-key-1.png"
        alt="Developer dashboard showing sandbox API key creation"
        style={{ borderRadius: '0.75rem' }}
      />
    </Frame>

    1. Log into your meCash Business dashboard
    2. From the sidebar, open **Developers â†’ API Keys**
    3. Click **Create a Key** (ensure the Sandbox environment is selected)
    4. Copy and securely store your API key
    
    <Tip>Keep your API key secure and never share it publicly.</Tip>
  </Accordion>
  <Accordion icon="arrows-split" title="Environment Setup">
    - **Production**: `https://api.me-cash.com` - For live transactions
    - **Sandbox**: `https://sandboxapi.me-cash.com` - For testing and development
    
    <Note>Use sandbox for testing your integration before going live.</Note>
  </Accordion>
</AccordionGroup>

### Step 3: Configure Security

<Accordion icon="shield" title="IP Whitelisting">
  <Frame>
    <img
      src="public/images/ipwhitelisitng1.png"
      alt="IP whitelisting configuration page listing approved IP addresses"
      style={{ borderRadius: '0.75rem' }}
    />
  </Frame>

  1. Go to **Developer** section in your dashboard
  2. Navigate to **IP Whitelisting**
  3. Add your server's IP addresses
  4. Save the configuration
  
  <Warning>API calls will be rejected from non-whitelisted IPs.</Warning>
</Accordion>

### Step 4: Make Your First API Call

<Accordion icon="code" title="Test API Connection">

  Try this simple request to test your setup:

  ```bash
  # Test with sandbox environment
  curl -X GET "https://sandboxapi.me-cash.com/v1/wallets" \
    -H "x-api-key: YOUR_API_KEY" \
    -H "Content-Type: application/json"
  ```

  <Tip>Replace `YOUR_SANDBOX_API_KEY` with your sandbox API key. Use sandbox for testing!</Tip>
</Accordion>
