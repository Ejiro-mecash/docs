# meCash Environment

meCash offers a Production environment for development and deployment.

The Production environment for processing real transactions with your users.

#### Production Base URL
`https://api.me-cash.com`

### Key Features:
- **Real Transactions:** Processes genuine payments and affects actual accounts.
- **Secure Operations:** Employs robust security measures to protect sensitive data.
- **High Availability:** Designed for reliability and uptime to support business-critical operations.

### How to Access:
- **Account Setup:** Ensure you have a verified [meCash production account](/account-creation.mdx).
- **API Keys:** Obtain [API keys](/authentication.mdx) from your meCash Developer dashboard.
